## 1) Gene: Coding & Noncoding
### 1a) Gene and Central Dogma
- A gene is a piece of information encoded in nucleic acids.
- For a long time, it's widely believed that genes are what encodes messenger RNA.
### 1b) Why do Humans Have So Few Genes?
- For human, the numer of genes encoding proteins is only 20000, which comprises the genome. 
- Ameoba have the largest genome among the all known organisms, which is 670 Gbp.
- 97%-98% of genome in human is not encoding for proteins, though 75% region are transcribed.
- Many ncRNA have been discovered in recent decades:
  >1. **Canonical ncRNAs**: rRNA, tRNA, snRNA, snoRNA, srpRNA
  >2. **Small ncRNAs**: miRNA, piRNA, siRNA
  >3. **Long ncRNAs**: MALAT1, HOTAIR

## 2) Method: Sequencing & Computing
### 2a) Revolution of Biology: Sequencing technologies
- Revolution of sequencing power was around 2007-2008 when NGS appeared.
- Big and HD (High-dimensional) data
### 2b) Steps of Bioinformatics
#### 1. Information
- Medical and/or biological data
#### 2. Model
- Mathematical and Probability Model
  > 1. **Regression Model: Linear ➡ Logistic Regression**
  > 2. **Tree Model: Decision Tree ➡ Random Forest**
  > 3. **Neural Network Model ➡ Deep Learning**
#### 3. Algorithm
- Computational Algorithm
  > - **Number Sorting Algorithm**
  > - **Dynamic Programming Algorithm** for blasting
- What is the difference between Model and Algorithm?
  > 1. **Algorithm** is the solution of a class of problems, is a part of the model building process, and also gives the model the ability to "think". It is a program written by a human and does not contain a mathematical model.
  > 2. A **model**, on the other hand, is a mathematical representation of a set of algorithms that contain both data and the concept of a task.
